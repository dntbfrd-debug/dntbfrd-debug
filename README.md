## 🚀 Основной проект

### ✨ [snbld resvap](https://github.com/dntbfrd-debug/snbld_pyside)
Самая продвинутая система макросов для Perfect World
- 🎯 Умные макросы с проверкой кулдаунов и дистанции
- 👁️ OCR распознавание через Tesseract
- 🎨 Детекция кастбара по пикселям
- 🔄 Автоматический ресвап экипировки
- ⚡ Qt Quick / QML интерфейс
- 🔐 Активация по HWID
