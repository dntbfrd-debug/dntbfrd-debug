## 🚀 Основной проект

### [snbld resvap](https://github.com/dntbfrd-debug/snbld_pyside)
Система макросов для MMORPG Perfect World на Python + PySide6 (QML)

- 🎮 Автоматизация игровых действий
- 🔍 OCR распознавание через Tesseract
- ⚡ Кастомный интерфейс на Qt Quick/QML
- 🎯 Точная система скиллов и баффов
